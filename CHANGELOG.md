# 更新记录

## V1.0 - 2026-05-02

- 建立高血脂风险分层预警与干预推荐系统 Web版。
- 实现 Flask 单服务架构，支持 Web 页面和 REST API 同服务运行。
- 实现信息录入、风险计算、风险结果展示和干预推荐四步流程。
- 实现三级风险分层预警，输出低风险、中风险和高风险结果。
- 实现异常风险因素识别，包括血脂指标、BMI、血尿酸、痰湿积分和活动能力。
- 实现患者画像分型，包括代谢异常主导型、肥胖并发型和功能受限型。
- 实现预算约束下的个体化干预方案优化和备选方案展示。
- 拆分 Web 运行依赖和可选机器学习训练依赖。
- 补充软件著作权申请说明、用户使用手册、软件设计说明书、测试报告和提交材料清单。
