## 软件概述

本软件基于《基于中西医融合信息的高血脂风险识别、分层预警与干预优化研究》（队伍编号 CMC2604725）的研究成果，面向中老年高血脂风险辅助评估场景，提供信息录入、三级风险分层、异常因素提示、患者画像分型和个体化干预方案推荐等功能。

系统采用 Web 单服务架构，后端使用 Python Flask 提供页面访问和 REST API 接口，前端使用 HTML5、Bootstrap 5 和 JavaScript 实现交互界面。用户启动服务后通过浏览器访问系统，无需额外部署前端服务器。

本软件仅用于科研、教学和健康管理辅助参考，不作为临床诊断或治疗依据。

## 功能模块说明

### 信息录入模块

采集用户基本信息、生化指标、中医体质积分、活动能力评分和干预预算，包括年龄、性别、身高、体重、TG、TC、HDL-C、LDL-C、血尿酸、痰湿体质积分、ADL 活动能力评分和 6 个月预算上限。

### 示例病例模块

系统内置低风险、中风险和高风险三类示例病例，用于教学演示、功能测试和软著截图准备。用户可一键填充示例数据，快速完成风险评估流程。

### 三级风险分层预警模块

基于论文提取的关键阈值，对输入指标进行异常识别与风险累计，输出低风险、中风险或高风险结果，并展示对应颜色、文字描述和异常因素列表。

关键阈值包括：

- 痰湿体质积分高风险阈值：60 分
- TG 高风险阈值：1.7 mmol/L
- TC 高风险阈值：6.2 mmol/L
- 活动能力高风险阈值：40 分及以下
- LDL-C、HDL-C、BMI、血尿酸等辅助阈值

### 患者画像分型模块

根据年龄、BMI、痰湿积分和活动能力对用户进行画像分型：

- 代谢异常主导型：血脂异常为主，体质和活动能力尚可
- 肥胖并发型：BMI 超标、痰湿体质明显并伴随代谢异常
- 功能受限型：活动能力下降、高龄或身体耐受度较弱

### 个体化干预优化模块

系统在预算和身体耐受度约束下，对中医调理等级、运动强度和运动频率进行组合筛选，按预期改善效果排序，输出最优方案和备选方案。

### 可视化与报告模块

系统提供指标与阈值对比图、方案对比卡片、评估报告预览、报告复制和浏览器打印功能。报告内容包括输入摘要、风险等级、异常因素、患者画像、干预建议、成本效果和免责声明。

### 可选特征识别模块

在安装可选机器学习依赖后，可使用 Lasso-Logistic、XGBoost-SHAP 和随机森林交叉验证完成关键风险因素识别，辅助验证论文中的核心特征结论。

## 核心算法说明

### 风险分层算法

```text
输入: 年龄、BMI、痰湿积分、TG、TC、HDL-C、LDL-C、血尿酸、活动能力评分
过程:
  1. 按论文阈值判断各指标异常程度
  2. 对偏高、过高、偏低或受限指标进行风险计分
  3. 根据总风险分数映射低、中、高三级风险
  4. 输出风险等级、异常因素和模型阈值说明
输出: 风险分层结果
```

### 干预组合优化算法

```text
输入: 患者画像类型、6个月预算上限
过程:
  1. 根据画像类型设置调理等级和运动强度约束
  2. 枚举全部调理和运动组合
  3. 过滤超过预算或不满足耐受度约束的组合
  4. 按预期效果排序，并在效果接近时优先选择成本更低方案
输出: 最优干预方案和备选方案
```
